m芯片Mac使用vagrant安装debian

发表于 2024-02-24 更新于 2024-02-25 阅读次数：

安装

下载VM Fusion

直接在官网下载或使用homebrew：

brew install --cask vmware-fusion

下载 vagrant

在这里选择macos下的arm64下载，或者：

brew tap hashicorp/tap

brew install hashicorp/tap/hashicorp-vagrant

下载Vagrant VMware Utility

在这里选择AMD64下载，或者：

brew install --cask vagrant-vmware-utility

下载vagrant plugin

vagrant plugin install vagrant-vmware-desktop

配置（以debian为例）

打开你的termina

新建文件夹(名字随意)并进入

mkdir vagrant

cd vagrant

初始化vagrant环境

vagrant init bento/debian-12-arm64

创建并运行虚拟机

vagrant up

vagrant ssh

就OK啦
